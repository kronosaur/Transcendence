Welcome to Transcendence!

Transcendence is a game of space combat and adventure in which you travel through the star systems of Human Space and encounter both friends and enemies. For more information on the game and to download the latest version, visit the Transcendence page online.

Intallation Instructions

Transcendence has been tested on Windows XP, Windows Vista, Windows 7, and Windows 8. To install, copy all the files in this ZIP file to a folder on your hard disk. Launch Transcendence.exe to start the game. You should have the following files:

    Readme.htm
    Transcendence.exe
    Transcendence.tdb
    TranscendenceIntro.mp3

Quick Start Guide

The [F1] key brings up help anytime during the game. Use the following keys to control your ship:
  • Arrow keys Move your ship. The left and right arrow keys turn your ship and the down arrow key thrusts forward.
  • [ctrl] key Fires the ship's primary weapon.
  • [shift] key Fires the ship's missile launcher. Your ship will not start out with a launcher. You must obtain one and install it.
  • [tab] key Selects the type of missile to fire.
  • [S] key Allows you to refuel your ship and use items in your cargo hold.
  • [M] key Shows a map of the entire star system.
  • [D] key Docks with a nearby station or wreck.
  • [A] key Engages the auto-pilot, which lets you travel faster.
  • [G] key Enters a stargate.
  • [U] key Uses an item in your cargo hold.
  • [W] key Selects a primary weapon to use. This is only useful if you have more than one primary weapon.
  • [T] key Selects the next enemy target. This only works if you have a targeting computer.
  • [I] key Invokes the powers of Domina.
  • [Q] key Communicates with your squadron.
  • [C] key Communicates with individual ships.
  • [B] key Enables/disables devices.
  • [N] key Shows stargate network (if galactic map installed).
  • [P] key Pauses the game.
  • [Esc] key Accesses the game menu.
